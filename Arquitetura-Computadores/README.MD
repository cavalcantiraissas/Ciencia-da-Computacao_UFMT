# Arquitetura de Computadores

Projetos e estudos realizados na disciplina de Arquitetura de Computadores, focados na interface hardware-software e no projeto de microprocessadores.

---

## Objetivos

Compreender a execução de programas em nível de hardware e o projeto do conjunto de instruções que define a interface fundamental do computador.

---

## Conteúdo Programático

- Modelo de Von Neumann e ciclo de execução de instruções
- Arquitetura do conjunto de instruções (ISA) RISC-V
- Programação em linguagem Assembly
- Microarquiteturas: Ciclo único, Multiciclo e Pipeline
- Hierarquia de memória e memória Cache
- Evolução de microprocessadores e lógica digital
