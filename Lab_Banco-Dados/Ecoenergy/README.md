# EcoEnergy - Sistema de Controle de Consumo de Energia

Projeto desenvolvido utilizando:

- Java 17
- Hibernate ORM
- JPA
- Maven
- Banco H2
- JPQL

## Funcionalidades

- Cadastro de registros de consumo
- Classificação por nível de consumo
- Persistência em banco relacional
- Consultas avançadas com JPQL
- Atualização de tarifas

## Como executar

1. Abrir o projeto na IDE
2. Atualizar dependências Maven
3. Executar Main.java

O Hibernate criará automaticamente a tabela no banco H2.
