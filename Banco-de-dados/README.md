# Banco de Dados

Repositório destinado ao estudo teórico e prático de Sistemas Gerenciadores de Banco de Dados (SGBD) e modelagem de dados relacionais.

## Informações Gerais
* Instituição: Universidade Federal de Mato Grosso
* Docente: Anthony Ferreira La Marca
* Período: 2025/2
* Carga Horária: 64 horas

## Objetivos
Fornecer base sólida para o projeto de bases de dados relacionais consistentes. O foco reside na aplicação de normas de integridade, normalização e proficiência em linguagem de consulta estruturada.

## Conteúdo Programático
* Arquitetura geral de um SGBD
* Modelagem Conceitual: Modelo Entidade Relacionamento (MER)
* Modelagem Lógica: Modelo Relacional e mapeamento ER para Relacional
* Teoria da Normalização e formas normais
* Álgebra Relacional e Cálculo Relacional
* Linguagem SQL: Comandos DDL (Definição) e DML (Manipulação)
