# Programação II

Projetos desenvolvidos em C++, com ênfase no paradigma de Orientação a Objetos e no uso de recursos avançados da linguagem.

## Informações Gerais
* Instituição: Universidade Federal de Mato Grosso
* Docente: Sandino Barros Jardim
* Período: 2024/1
* Carga Horária: 64 horas

## Objetivos
Capacitar o aluno no desenvolvimento de sistemas robustos utilizando C++, aplicando conceitos de encapsulamento, herança e polimorfismo.

## Conteúdo Programático
* Alocação dinâmica de memória e ponteiros inteligentes
* Classes, Atributos e Métodos
* Encapsulamento, Construtores e Destrutores
* Herança, Polimorfismo e Funções Virtuais
* Sobrecarga de operadores
* Tratamento de Exceções e Programação Genérica (Templates)
* Biblioteca de Modelos Padrão (STL)

## Bibliografia Principal
* DEITEL, H. M. C++ como Programar
* STROUSTRUP, B. Programming: Principles and Practice Using C++
* MIZRAHI, V. V. Treinamento em Linguagem C++
