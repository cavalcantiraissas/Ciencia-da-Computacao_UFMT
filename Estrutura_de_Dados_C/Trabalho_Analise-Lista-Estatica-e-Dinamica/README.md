# Trabalho - Análise de Lista Estática e Dinâmica

Trabalho prático comparativo entre as estratégias de alocação estática e dinâmica na implementação de listas lineares em C. O objetivo é evidenciar as diferenças de comportamento, vantagens e limitações de cada abordagem em termos de uso de memória e desempenho das operações.

---

## Estruturas Implementadas

| Arquivo | Descrição |
|---|---|
| `lista_estatica.c` | Lista com alocação estática (vetor de tamanho fixo) |
| `main_estatica.c` | Programa principal para teste da lista estática |
| `lista_dinamica.c` | Lista simplesmente encadeada com alocação dinâmica (`malloc`/`free`) |
| `main_dinamica.c` | Programa principal para teste da lista dinâmica |
| `lista_duplamente_encadeada.c` | Lista duplamente encadeada com alocação dinâmica |
| `main_duplamente_encadeada.c` | Programa principal para teste da lista duplamente encadeada |
