# Atividades de Classe - Estrutura de Dados 

Exercícios e atividades práticas realizadas durante as aulas de Estrutura de Dados. O conteúdo cobre desde exercícios introdutórios de manipulação de dados até implementações completas de estruturas lineares e algoritmos de ordenação.

---

## Conteúdo

### Subpastas

| Pasta | Descrição |
|---|---|
| [Fila_Alocacao-Dinamica-Encadeada](./Fila_Alocacao-Dinamica-Encadeada) | Implementação de fila com alocação dinâmica via lista encadeada |
| [Fila_Pilha_Alocacao-Estatica](./Fila_Pilha_Alocacao-Estatica) | Implementação de fila e pilha com alocação estática (vetor) |
| [Heap-Sort](./Heap-Sort) | Implementação do algoritmo de ordenação Heap Sort |
| [Implementacao_Lista](./Implementacao_Lista) | Implementação de lista linear com operações de inserção, remoção e busca |
| [Insercao_Lista-Duplamente-Encadeada](./Insercao_Lista-Duplamente-Encadeada) | Inserção e manipulação em lista duplamente encadeada circular |

### Exercícios Avulsos

| Arquivo | Descrição |
|---|---|
| `ex1_Leitura-de-Media.c` | Leitura de valores e cálculo de média |
| `ex2_Leitura-de-letras.c` | Leitura e contagem de caracteres |
| `ex3_Criacao-Sequencia` | Criação e impressão de sequência numérica |
| `ex4_Multiplicacao.c` | Operações de multiplicação com controle de entrada |
| `ex5_Divisao-por-tres.c` | Filtragem de divisíveis por três em uma sequência |
| `ex_Cadastro-de-alunos.c` | Cadastro de alunos com uso de registros (`struct`) |
| `ex_Registros.c` | Manipulação de registros e acesso a campos de struct |
| `Ordenacao_Merge-Sort.c` | Implementação do algoritmo Merge Sort |
