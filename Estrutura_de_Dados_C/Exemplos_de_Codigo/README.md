# Exemplos de Código - Estrutura de Dados I

Biblioteca de implementações de referência em C, cobrindo as principais estruturas de dados e conceitos estudados na disciplina. Os arquivos servem como base de consulta e estudo para as demais atividades e trabalhos.

---

## Implementações Disponíveis

### Estruturas Lineares

| Arquivo | Descrição |
|---|---|
| `Manipulando_vetor.c` | Operações básicas sobre vetores: inserção, remoção e busca |
| `Manipulando_Matriz.c` | Operações sobre matrizes bidimensionais |
| `Manipulacao_Registros.c` | Manipulação de structs e registros em C |
| `Implementacao_PILHA.c` | Pilha com alocação estática |
| `Pilha_com_Encadeamento.c` | Pilha com alocação dinâmica via lista encadeada |
| `Implementacao_FILA.c` | Fila com alocação estática |
| `Fila-com-encadeamento.c` | Fila com alocação dinâmica via lista encadeada |
| `Lista_com_alocacao_dinamica.c` | Lista simplesmente encadeada com alocação dinâmica |
| `Lista_duplamente-encadeada.c` | Lista duplamente encadeada |
| `Lista_circular_alocacao_dinamica.c` | Lista circular com alocação dinâmica |

### Estruturas Não-Lineares

| Arquivo | Descrição |
|---|---|
| `Arvore_Binaria.c` | Árvore binária de busca com inserção, remoção e percursos |
| `Arvore_com_multiplos_filhos.c` | Árvore genérica com número variável de filhos por nó |
| `Heap.c` | Implementação de heap máximo com operações de inserção e remoção |
| `Tabela_Hash.c` | Tabela hash com tratamento de colisões |

### Outros

| Arquivo | Descrição |
|---|---|
| `Ponteiros.c` | Fundamentos de ponteiros: aritmética, passagem por referência e alocação dinâmica |
| `Geracao-de-numeros-pseudoaleatorios.c` | Geração de sequências pseudoaleatórias para testes |

---

## Como Compilar

```bash
gcc nome_do_arquivo.c -o programa
./programa
```
