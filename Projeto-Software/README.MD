# Projeto de Software

Este diretório armazena a documentação, diagramas e implementações referentes à disciplina de Projeto de Software, focada na aplicação de engenharia de software para sistemas escaláveis.

## Informações Gerais
* Instituição: Universidade Federal de Mato Grosso
* Docente: Thiago Pereira da Silva
* Período: 2026/1
* Carga Horária: 64 horas (Prática)

## Objetivos
Proporcionar a capacidade de projetar sistemas de software robustos e de fácil manutenção, aplicando padrões de design, princípios de arquitetura e boas práticas de engenharia

## Conteúdo Programático
* Princípios de Design: SOLID, DRY, KISS, YAGNI, Coesão e Acoplamento
* Padrões de Projeto (Design Patterns): Criacionais, Estruturais e Comportamentais 
* Estilos Arquiteturais: Camadas (MVC), Arquitetura Hexagonal e Microsserviços 
* Modelagem com UML e versionamento com Git/GitHub [cite: 531]
* Metodologias Ágeis: Scrum e Kanban
