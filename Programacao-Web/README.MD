# Programação IV

Foco no desenvolvimento de aplicações Web Full Stack, integrando tecnologias de frontend, backend e práticas modernas de engenharia de software.

## Informações Gerais
* Instituição: Universidade Federal de Mato Grosso
* Docente: Sandino Barros Jardim
* Período: 2025/1
* Carga Horária: 64 horas

## Objetivos
Capacitar para a criação de sistemas distribuídos e responsivos, unindo interfaces interativas a lógicas de negócio complexas e persistência em nuvem.

## Conteúdo Programático
* Arquitetura Web: Requisições HTTP, Servidores e Clientes
* Desenvolvimento Frontend: HTML5, CSS3 e JavaScript
* Desenvolvimento Backend: Programação Orientada a Objetos com PHP
* Versionamento de código com Git e fluxos de trabalho colaborativos
* Testes automatizados e garantia de qualidade
* Segurança em aplicações Web e proteção contra vulnerabilidades comuns

## Bibliografia Principal

* NIDERAUER, Juliano. Desenvolvendo websites com PHP
