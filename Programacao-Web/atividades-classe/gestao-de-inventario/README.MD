# Objetivo
Refatorar o sistema de gestão de inventário para uma estrutura de projeto modu- lar e profissional. Em vez de um único ficheiro, vamos separar as responsabilida- des em múltiplos ficheiros, usando o sistema de Módulos ES6 (import/export). Esta abordagem torna o código mais organizado, reutilizável e fácil de manter.
