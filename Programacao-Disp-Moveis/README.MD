# Programação para Dispositivos Móveis

Repositório dedicado ao armazenamento de projetos de desenvolvimento mobile realizados durante a disciplina optativa de Programação para Dispositivos Móveis.

## Informações Gerais
* Instituição: Universidade Federal de Mato Grosso
* Docente: José Marques Pessoa
* Período: 2026/1
* Carga Horária: 64 horas

## Objetivos
Capacitar o estudante na compreensão de infraestruturas e frameworks para o desenvolvimento de aplicativos móveis, com foco prático nos ambientes Android Studio e Lazarus Android Module Wizard

## Conteúdo Programático
* Introdução a infraestruturas e frameworks mobile 
* Arquitetura de desenvolvimento em Android Studio com Kotlin e Flutter
* Arquitetura LAMW (Lazarus Android Module Wizard) 
* Comunicação cliente servidor e persistência de dados 
* Design de interfaces (UX/UI) para dispositivos móveis
